## Run Locally 🖥

Clone the project

```bash
  git clone https://github.com/Akbarjon03/instagram-2.0
```

Go to the project directory

```bash
  cd instagram
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
